Advanced C++

Syllabus

Course Description

Advanced C++ is a first year computer science which assumes an inroductory knowledge of C++. The course is intended to be taken concurrently with Data Structures but can be taken independently. Essentially the course covers the implementation of the algorithms discussed in Data Structures. These algorithms include but are not limited to:

  • Linked Lists
  • LIFO / FIFO
  • Binary Search Trees
  • Priority Queues
  • Red Black Trees
  • AVL Tree

The course also covers techniques and advanced topics in C++:

  • Object oriented programming design
  • Inheritance
  • Virtual Classes
  • Template Classes
  • Standard Template Library
  • Exception Handling
  • Operator Overloading
  • Advanced Sorting algorithms
  • Advanced Recursive Algorithms

Prerequisites

  • Introduction to Computer Science in C++

Text Books

  • C++ How to Program (5th Edition) (Deitel and Deitel)
  • Algorithms in C++, Parts 1-4: Fundamentals, Data Structure, Sorting, Searching
