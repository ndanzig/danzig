    Points Unit Description
    25 Project 1 TFTP server (small files
    25 Project 2 TFTP server (regular files)
    25 Project 3 TCP-based TFTP-like server (more than one client at a time)
    10 Test 1 converting text to socket address

    (nochachut)    		 These are the points you get if you have attended at least 7 sessions. Those who are exempt from this requirement will not be penalized.

    1. You get these points if you turn all your projects and do the test on time.

    3. Being late for the test means losing 1 point.
