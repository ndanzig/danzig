Here is the header of a function that host name (or IP address) and the port number (or service name) and fills in the socket address structure:

void str_to_sockaddr (char *host,  char *port,  struct sockaddr_in *s)

You need to write the C code for this function.  I'm mainly interested in seeing that you know which functions to call, and what the fields of the sockaddr_in structure are.   Note that there might be some fields of the structure that you have to fill in that do not depend upon which host or port you use.
