Real vs. Protected Mode
•Protected mode is the mode that the computer runs in when it has to support multiple users. UNIX and XP run only in protected mode. A regular user can’t access the I/O ports. •Real mode (or single-user mode, or priveleged-mode) is the mode used by the OS to access I/O devices. DOS runs in real mode. So we  will use a Virtual PC to run DOS in order to run our programs. •inp and outp work when compiled with BorlandC, the Visual C++ versions (_inp and _outp) might create problems, try it out.
