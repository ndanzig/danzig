The Line Status Register (LSR)
•The LSR shows the current status of communication, errors in transmission will be written into the register. •Bit 0: Data Ready (DR) indicator. When set, it indicates that a byte is ready to be read from the RBR. Reading from the RBR sets the DR bit to 0. •Bit 1: Overrun Error (OE) indicator. When set indicates that a new byte has been received before the current byte in the RBR has been read. The OE is reset (set to 0) when the LSR is read. •Bit 2: Parity Error (PE) indicator. When set indicates that a parity error has occurred. Is reset by reading the LSR. •Bit 3: Framing Error (FE) indicator. Is set whenever the received word doesn't have a valid stop bit. The stop bit following the last data bit or parity is detected as a 0 instead of a 1. •Bit 4: Break Interrupt (BI) indicator. Is set whenever the RD line is held in the space state for longer than it takes to send a word.
