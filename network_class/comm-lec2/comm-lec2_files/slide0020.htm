The Control Lines
•DTR (Data Terminal Ready) - Indicates to the modem that the UART is ready to communicate. •DSR (Data Set Ready) - Indicates to the UART that modem is ready to communicate. •RTS (Request To Send) - This line informs the modem that the UART has data to be sent. •CTS (Clear To Send) - Indicates that the modem is ready to exchange data.
