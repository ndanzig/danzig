Accessing the Registers

  • The port addresses for the COMs are defined in the BIOS (Basic I/O System) ROM (Read Only Memory) from addresses 0x400 to 0x408.

  • OK. So how do we access the registers?

  • There is a C interface to the IN and OUT instructions: int inp(unsigned short port); // read a byte from the port int outp(unsigned short port, int val); // write a byte to the port

  • Using these two instruction is is possible to access the registers defined in the previous slides. For instance to read the LCR we have to write: int val; val = inp(0x3F8 + 3); // or inp(0x3FB);

  • Look at the functions descriptions in the help manuals of Visual C++ or BorlandC.
