GENERAL NOTES

pseudo-code
Some of the documentation and examples are in pseudo-code instead of real C code, because the real C code is very verbose, and I thought it would be easier to understand the socket functions in pseudo-code.
versions
Some of the C code examples were compiled on hosts running the SunOS 4 version of UNIX (e.g., hosts 'avoda' and 'beitza'). Others have been compiled under Linux (Red Hat distribution, versions 6.0, 6.1, 6.2, or 7). The PCs in the UNIX lab currently run Red Hat 7. These are the PCs in the UNIX lab. Their names are shana200-shana220.

Some of the definitions and structures are different between the two types of systems, so what compiles on one might not compile on the other. But you can easily adapt the examples.

All code in this site been compiled with the GNU C (not C++) compiler (command 'gcc').

For several reasons, you should compile all code you write on Linux (unless told otherwise).
