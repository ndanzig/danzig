 2. HTML Introduction
	Material:
	* HTML tags are used to mark-up HTML  elements
	* HTML tags are surrounded by the two characters < and >
    	* The surrounding characters are called angle brackets
    	* HTML tags normally come in pairs like <b> and </b>
    	* The first tag in a pair is the start tag, the second tag is 	 
the end tag
    	* The text between the start and end tags is the element content
    	* HTML tags are not case sensitive, <b> means the same as <B>

	<html>,<head>,<title>

	Tag Attributes
	Tags can have attributes. Attributes can provide additional
	information about the HTML elements on your page.
	This tag defines the body element of your HTML page: <body>. 	With an
added bgcolor attribute, you can tell the browser that the 	background
color of your page should be red, like this: 
	<body bgcolor="red">.
	Attributes always come in name/value pairs like this: 	name="value".
	Attributes are always added to the start tag of an HTML element.

	<body>
